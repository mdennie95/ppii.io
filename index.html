OUR PLAN

OUR STEPS TO SUCCESS

  • DESIGN
  • TAG SYSTEM
  • DESCRIPTIONS
  • BENEFITS

We have designed a digital database containing a file on each object. These files contain: the object name, high-quality photographs of the object, a record of its current location, a record of where it should be when in storage, a description to add more detail to the object, and any other data mentioned elsewhere (such as 3D scans).

The database uses artificial intelligence to generate a list of describing words for each object called "tags". The tags make it easier to find specific objects in the database, as someone searching the database can easily find all objects with certain tags. AI generation, in this instance, increases the speed at which tags can be generated significantly. This also removes the need for staff to be trained to add tags themselves.

The description within the file allows for museum staff to manually add important context to an object as required. In this way someone viewing the file later will be able to see context such as: specific safety instructions, the object’s function, and culturally sensitive considerations. This can be added or updated after the object has been put into the database allowing for information to be added or edited in the future. Artificial intelligence will also be used to generate more tags from this description. This allows for objects to be found even more easily in the database.

This database system will be easy to implement, in spite of the museum’s large collection. This is both due to how simple and standardised it is. This also means it is easy to maintain as further object files can be added and edited with little effort. Furthermore, it is easy for staff to use due to the tag system allowing for easy navigation of the vast database even without necessarily knowing what an object is.

We have integrated articifical intelligence and machine learning into our solution. We will be using Optical Character Recognition to scan through and extract text that is in the images, this includes labels with the artefacts and in text included in the artefact.

Considering the size of the collection, we recommend separating the artefacts into four different phases to ensure we maximise efficiency while still ensuring all the requirements are met. These phases are based on the different kind of objects and the difficulty level involved.

Virual tours will allow the museum to be accessible worldwide and create learning opportunities that were never achievable before. With the addition of QR codes around the museum, the in person experience will also be elevated.

04

ADDED FEATURE

SENSOR USAGE

Sensors can be used in order to better connect the database with the objects in the museum’s collection. A System is being implemented which allows staff to scan a QR code on an object with their phone and then be linked directly to its file in the database. Another use of sensors is in monitoring the state of objects. Temperature sensors monitor how cold each of the 11 freezers are and display that on the files of the objects within them. Various sensors are also being used in determining the quality of the preservation fluid within liquid preserved objects, allowing for them to be monitored without retrieving the object manually, a time consuming and potentially dangerous process.
